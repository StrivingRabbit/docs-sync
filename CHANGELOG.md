## 1.0.0

### Publish Changes

- 🔄 **多源同步**: 从多个 Git 仓库拉取文档
- 📁 **本地源支持**: 直接使用本地目录作为文档源（无需拷贝，实时反映变化）
- 📝 **包含系统**: 使用 `@include` 指令引用和包含已同步源的内容
- 🎯 **站点过滤**: 根据站点配置有条件地包含/排除内容
- 📊 **依赖追踪**: 使用依赖图进行增量编译
- 👀 **监视模式**: 当源文件更改时自动重新编译（支持本地源监视）
- 🔍 **预演模式**: 预览更改而不写入文件
- ⚙️ **灵活配置**: 支持自定义配置文件路径
